## 添加新应用

编辑 `index.html` 中的 `APPS` 数组：

```js
const APPS = [
  {
    title: '应用名称',
    description: '简短描述',
    href: 'your-app.html',
    tags: ['标签'],
    date: '2026-03-04',
  },
]
```
